# Бот для генерации контента НКО

Этот Telegram-бот помогает некоммерческим организациям (НКО) быстро создавать качественный контент для социальных сетей.

## Возможности бота

- **Создание текстов** — генерация постов, призывов к действию и других текстов на основе вашей идеи
- **Генерация изображений** — создание визуального контента по текстовому описанию
- **Редактирование текста** — проверка и улучшение уже существующих текстов
- **Создание контент-плана** — составление плана публикаций на заданный период
- **Персонализация** — использование информации об НКО для более точной генерации контента

## Как использовать

1. Запустите бота командой /start
2. Заполните информацию о вашей организации (необязательно, но рекомендуется)
3. Используйте кнопки меню для выбора нужного действия
4. Введите вашу идею или выберите параметры для генерации

Бот работает на базе GigaChat и автоматически создает качественный контент, готовый к публикации.
